# 2. 개인적으로 배운 것/리마인드

🌿 **tmux 커맨드 정리**

```py
# 새로운 tmux창 생성
tmux new -s new_session

# 실행 중인 tmux 창 확인
tmux ls

# 실행 중인 tmux 창 접속하기
tmux attach -t new_session

# 접속한 tmux 창에서 빠져나오기
(ctrl + b), d

# tmux 창 내에서 스크롤
(ctrl + b), [
# 스크롤 모드 종료
esc
```
